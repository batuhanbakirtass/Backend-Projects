## **VAT Calculator**

This Java program calculates the VAT (Value-Added Tax) amount and the total price including VAT based on user input. It applies an **18% VAT rate** for amounts between **0 and 1000 TL** (inclusive) and an **8% VAT rate** for amounts over **1000 TL**. The program displays the price without VAT, VAT rate, VAT amount, and total price. 

To run the program, compile it with `javac VATCalculator.java` and execute it using `java VATCalculator`. Enter the amount when prompted to see the calculated results. This simple, user-friendly program is a great example of dynamic VAT calculation in Java.
